Tag Pre

Agora utilizaremos em conjunto a tag code com a tag pre. A tag pre herda a indentação, logo tudo será considerado. Por exemplo:

        num = int(imput('Digite um número'))
        if num % 2 == 0:
            print(f'0 número {num} é PAR')
        else:
            print(f'0 número {num} é ÍMPAR')
        print('Fim do programa')

apesar de não ser a forma recomendada, podemos curiosamente usar a tag pre para substituir a tag br, para quebrar linhas. exemplo:

        teste


        teste

Citações de blocos, uma tag blockquote

Esta tag aparece deslocada horizontalmente á direita e é direcionada para um determinado autor. Além disso, podemos adicionar um parâmetro "cite", que apesar de não ser exibido no terminal, serve como mecanismo de busca. Pode ser colocado por exemplo, link de venda ou mesmo garantir os direitos daqueles que produzem conteúdos
