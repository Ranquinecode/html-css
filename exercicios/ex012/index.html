Um vídeo em hospedagem local

Este vídeo está hospedado no meu próprio servidor

Foi adicionado os atributos:

  • controls: para fazer aparecer os botoes de controle
  • loop: para fazer loop infinito
  • preload=metadata: para carregar apenas as informações sem fazer o download do arquivo ao carregar a página
  • width=560: para determinar a largura do tamanho do video
  • height=315: para determinar a altura do tamanho do video
  • poster: para carregar uma imagem para ser exibida como thumbmail do video.
